关于const关键词的理： - 数据类型修饰符 - 类型修饰符如果有多个关键字 它们的出现不影响它们对变量的限制 eg. int i，short j - const int a int const a 完全一样 都是一个const类型的变量 - int *a[10] 因为[] 优先级高于 * 因此是一个数组 每个成员为一个指针指向int - int (*a)[10] ()优先级高，所以首先是一个指针， 然后这个指针指向一个数组 - const int *a 首先 * 优先级高，所以是一个指针，指向一个const int 数据 所以这个int 数据不可修改 - int * const a 首先a 是一个const 类型的指针，指向一个int值，a不可修改 总之这个只是基础概念的认识，把优先级知道基本就可以理解了。也包括函数指针，指针函数这种绕人的头疼概念
